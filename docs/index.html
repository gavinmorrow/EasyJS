Overview

What is EasyJS?

EasyJS is a JavaScript library for simplifying your code.

Try EasyJS

To try EasyJS, just add it to your page with just a few lines of code: <script type="module"> import EasyJSVersion from "http://gavinmorrow.com/easyjs/"; // You can set the version number (below) to any valid major version. const EasyJS = new EasyJSVersion(1); // This is the actual EasyJS interface. // Your code here! </script> You can also import individual sections or functions/APIs, but there will be more on that later.

Canvas

Try Canvas!

To try the EasyJS Canvas API, just add it to your page with just a few lines of code: <script type="module"> import Canvas from "http://gavinmorrow.com/easyjs/1/canvas/"; // You can set the version number (above) to any valid major version. // Your code here! </script>

Line Graph

The Line Graph API helps you make a line graph on the HTML canvas. To use it, you can import the rest of the Canvas interface, or just add this line: import LineGraph from "http://gavinmorrow.com/easyjs/1/canvas/lineGraph/;
