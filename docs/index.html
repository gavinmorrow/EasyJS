What is EasyJS?

EasyJS is a JavaScript library for simplifying your code.

Try EasyJS

To try EasyJS, just add it to your page with just a few lines of code: 
<script type="module">
import EasyJSVersion from "http://gavinmorrow.com/easyjs/";

// You can set the version number (below) to any valid major version. 
const EasyJS = new EasyJSVersion(1);

// Your code here!
</script>
You can also import individual sections or functions/APIs, but there will be more on that later.
